# 03 - 硬件设计

## 模块概述

消费电子产品硬件设计全流程：[[schematic-design|原理图]]设计、[[pcb-layout|PCB]] Layout、打板制造、[[pcba-process|BOM]] 管理。

## 目录结构

```
03-hardware/
├── schematic/      # 原理图设计
├── pcb/            # PCB Layout 与设计规则
├── manufacturing/  # 打板、焊接、组装
└── bom/            # BOM 管理与元器件选型
```

## 核心知识领域

### 1. 原理图设计

#### 设计流程
```
需求分析 → 系统框图 → 芯片选型 → 原理图绘制 → 仿真验证 → 评审 → 输出
```

#### 关键电路模块
- **电源系统**：LDO / DC-DC / 充电管理 / 电源时序
- **时钟系统**：晶振 / PLL / 时钟树 / 时钟分配
- **复位系统**：上电复位 / 手动复位 / 看门狗 / BOD
- **调试接口**：JTAG / SWD / UART Console
- **存储系统**：Flash / EEPROM / SD 卡 / eMMC
- **传感器接口**：ADC / I2C / SPI / 模拟前端

#### EDA 工具对比
| 工具 | 特点 | 适用 |
|------|------|------|
| Altium Designer | 功能全面、企业级 | 中大型团队 |
| KiCad | 开源免费 | 个人/小团队 |
| OrCAD / Allegro | 仿真强大 | 高速设计 |
| 立创 EDA | 国产、云端 | 快速原型 |
| Eagle | 轻量 | 小型项目 |

### 2. PCB 设计

#### Layout 关键原则
- **叠层设计**：2/4/6/8 层板选择与层叠规划
- **阻抗控制**：单端 50Ω / 差分 90Ω/100Ω
- **电源完整性**：[[schematic-design|去耦电容]]放置、电源平面分割
- **信号完整性**：走线等长、差分对、蛇形线
- **[[env-testing|EMC]] 设计**：回流路径、屏蔽、滤波

#### 设计规则检查 (DRC)
- 线宽/线距最小值
- 过孔尺寸与间距
- 铺铜间距
- 丝印清晰度
- 安装孔定位

### 3. 打板制造

#### 制造流程
```
Gerber 输出 → 工程审核 → 开料 → 钻孔 → 沉铜 → 图形转移 →
蚀刻 → 阻焊 → 丝印 → 表面处理 → 测试 → 成品
```

#### 表面处理
| 工艺 | 特点 | 成本 |
|------|------|------|
| HASL (喷锡) | 经济实惠 | 低 |
| ENIG (沉金) | 平整、可焊接性好 | 中 |
| OSP | 环保、成本低 | 低 |
| 沉银 | 高频性能好 | 中 |
| 电镀硬金 | 耐磨、金手指 | 高 |

#### 打板参数速查
| 参数 | 常规 | 高端 |
|------|------|------|
| 最小线宽 | 6mil | 3mil |
| 最小线距 | 6mil | 3mil |
| 最小过孔 | 0.3mm | 0.15mm |
| 层数 | 1-16 | 20+ |
| 板厚 | 0.4-3.2mm | 定制 |

### 4. BOM 管理

#### BOM 结构
```
Level 0: 成品
├── Level 1: PCBA
│   ├── Level 2: 主芯片
│   ├── Level 2: 被动元件
│   ├── Level 2: 连接器
│   └── Level 2: 结构件
├── Level 1: 电池
├── Level 1: 外壳
└── Level 1: 包装
```

#### 选型考虑
- 供货稳定性（生命周期、多源供应）
- 成本（批量价格、替代料）
- 性能参数（温度范围、精度）
- 封装（可制造性、散热）
- 认证（RoHS、REACH）

## 评审检查清单

- [ ] 电源树完整性与电流裕量
- [ ] 所有引脚连接正确（NC 引脚处理）
- [ ] 去耦电容数量与位置
- [ ] 晶振负载电容匹配
- [ ] [[schematic-design|ESD]] 保护器件放置
- [ ] 连接器引脚定义文档
- [ ] 测试点覆盖关键信号
- [ ] 丝印标注清晰无歧义
