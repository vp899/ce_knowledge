## 核心内容

### 1. 为什么需要 FOC

```
六步换向的问题:
├── 转矩脉动大 (±15%)
├── 效率低 (80-85%)
├── 噪音大
└── 低速性能差

FOC 的优势:
├── 转矩脉动小 (<1%)
├── 效率高 (95%+)
├── 噪音低
├── 低速性能好
└── 精确速度/转矩控制
```

### 2. 坐标变换

### Clark 变换 (3相→2相静止)
```
将三相电流 (ia, ib, ic) 转换为两相静止坐标 (iα, iβ):

  iα = ia
  iβ = (ia + 2·ib) / √3

物理含义:
  α 轴: 与 A 相同方向
  β 轴: 超前 α 轴 90°

  β ↑
    │   ╱ ib
    │  ╱
    │ ╱ 30°
    └──────→ α
         ia

从三相到两相, 减少了一个变量
```

### Park 变换 (2相静止→2相旋转)
```
将两相静止坐标 (iα, iβ) 转换为与转子同步旋转的坐标 (id, iq):

  id =  iα·cosθ + iβ·sinθ
  iq = -iα·sinθ + iβ·cosθ

θ = 转子电角度 (从编码器/霍尔传感器获取)

物理含义:
  d 轴: 与转子磁极同方向 (直轴)
  q 轴: 超前 d 轴 90° (交轴)

  iq → 产生转矩 (这是我们想要控制的)
  id → 产生磁通 (通常设为 0, 最大效率)

关键洞察:
  通过 Park 变换, 将交流量变成了直流量!
  → 可以用简单的 PI 控制器
```

### 4. SVPWM 详解

### 电压矢量
```
8 个基本电压矢量:

  V0(000) ─── 零矢量
  V1(100) ─── 0°
  V2(110) ─── 60°
  V3(010) ─── 120°
  V4(011) ─── 180°
  V5(001) ─── 240°
  V6(101) ─── 300°
  V7(111) ─── 零矢量

  V3(010)
    │
    │
V4(011)────O────V2(110)
    │      │
    │      │
  V5(001)  V1(100)
    │
  V6(101)

任意电压矢量 = 相邻两个基本矢量 + 零矢量 的时间加权
```

### 扇区判断与作用时间
```c
// SVPWM 实现
void svpwm(float valpha, float vbeta, float vbus,
           float *da, float *db, float *dc) {
    float ts = 1.0f;  // 归一化周期
    
    // 扇区判断
    float u1 = vbeta;
    float u2 = 0.866f*valpha - 0.5f*vbeta;
    float u3 = -0.866f*valpha - 0.5f*vbeta;
    
    int sector = 0;
    if(u1 > 0) sector |= 1;
    if(u2 > 0) sector |= 2;
    if(u3 > 0) sector |= 4;
    
    // 计算作用时间
    float t1, t2;
    float k = 1.732f * ts / vbus;
    
    switch(sector) {
    case 1: t1 = k*(0.866f*valpha - 0.5f*vbeta);
            t2 = k*vbeta; break;
    case 2: t1 = k*(-0.866f*valpha + 0.5f*vbeta);
            t2 = k*(0.866f*valpha + 0.5f*vbeta); break;
    // ... 其他扇区
    }
    
    // 过调制处理
    if(t1 + t2 > ts) {
        float r = ts / (t1 + t2);
        t1 *= r; t2 *= r;
    }
    
    float t0 = ts - t1 - t2;
```
