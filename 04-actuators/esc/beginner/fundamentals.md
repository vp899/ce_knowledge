## 核心内容

### 1. 无刷电机原理

### 有刷 vs 无刷
```
有刷电机 (DC Motor):
├── 结构: 定子(磁铁) + 转子(线圈) + 电刷+换向器
├── 优点: 简单, 便宜, 易控制
├── 缺点: 电刷磨损, 火花, 效率低
└── 寿命: 1000-3000 小时

无刷电机 (BLDC):
├── 结构: 定子(线圈) + 转子(磁铁) ← 无电刷!
├── 优点: 高效, 长寿命, 高功率密度
├── 缺点: 需要电子换向 (ESC)
└── 寿命: 10000+ 小时
```

### 无刷电机结构
```
外转子结构 (无人机常用):

        N   S   N   S
        ┌─────────────┐
        │  永磁体      │ ← 转子 (旋转)
        │  ┌───────┐  │
        │  │ 线圈   │  │ ← 定子 (固定)
        │  │ A B C  │  │
        │  └───────┘  │
        └─────────────┘

三相线圈: A, B, C (互隔 120°)
永磁体: N, S 交替排列

KV 值 (RPM/Volt):
├── KV = 空载转速 / 电压
├── 低 KV (100-300): 大扭矩, 低转速, 大桨
├── 高 KV (1000-3000): 小扭矩, 高转速, 小桨
└── 选型: 根据桨叶大小和负载选择
```

### 2. ESC (电子调速器) 基础

### ESC 功能
```
ESC = 电子换向器 + 速度控制器

功能:
├── 电子换向: 按顺序给三相通电
├── 速度控制: 根据油门信号调节转速
├── 保护: 过流/过温/低压保护
└── 回馈: 转速/温度/电流反馈

输入: PWM 信号 (油门)
输出: 三相交流 (驱动电机)
```

### PWM 油门信号
```
标准 PWM 信号:
├── 频率: 50Hz (20ms 周期)
├── 脉宽: 1000μs (停转) ~ 2000μs (全速)
├── 中位: 1500μs (半速)

  1000μs ─────── 1500μs ─────── 2000μs
  停转           半速           全速

DShot 数字信号 (更先进):
├── DShot300: 300kbit/s
├── DShot600: 600kbit/s
├── 分辨率: 2048 级
├── 无需校准
└── 支持双向 (回传转速)
```

### 3. 第一个电机控制程序

```c
// Arduino + ESC 基础控制
#include <Servo.h>

Servo esc;

void setup() {
    esc.attach(9, 1000, 2000);  // 引脚 9
    
    // ESC 校准 (首次使用)
    esc.writeMicroseconds(2000);  // 最大油门
    delay(3000);
    esc.writeMicroseconds(1000);  // 最小油门
    delay(3000);
    
    // 现在 ESC 已校准, 可以正常使用
}

void loop() {
    // 逐渐加速
    for(int us = 1000; us <= 2000; us += 10) {
        esc.writeMicroseconds(us);
        delay(50);
    }
    
    // 逐渐减速
    for(int us = 2000; us >= 1000; us -= 10) {
        esc.writeMicroseconds(us);
        delay(50);
    }
}
```

### 4. 安全注意事项

```
⚠️ 无刷电机非常危险!

安全规则:
├── 首次测试时卸下桨叶!
├── 从小油门开始, 逐渐增加
├── 远离旋转部件
├── 确保电池电压正确
├── 检查电机转向是否正确
└── 急停开关随时可用

常见错误:
├── 接线错误 → 电机反转 (交换任意两相)
├── 油门过大 → 电机过热/烧毁
├── 堵转 → 大电流, ESC 烧毁
└── 电池接反 → ESC 烧毁 (有防反接保护更好)

## 常见问题

### Q1: 电机不转怎么办？

**A**: 检查 ESC 供电是否正常；确认油门信号范围 (1000-2000μs)；检查电机线序是否正确；尝试重新校准 ESC。

### Q2: 电机转向反了怎么办？

**A**: 交换电机三根线中的任意两根，转向就会反转。
