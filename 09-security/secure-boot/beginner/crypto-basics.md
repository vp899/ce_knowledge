## 核心内容

### 1. 哈希 (Hash)

```
哈希: 任意长度 → 固定长度 (单向)

常用算法:
├── MD5: 128-bit, 已不安全
├── SHA-1: 160-bit, 已不安全
├── SHA-256: 256-bit, 安全
└── SHA-512: 512-bit, 安全

用途:
├── 密码存储: 存哈希, 不存明文
├── 完整性校验: 文件 SHA256
└── 数字签名: 签名哈希值
```

### 2. 对称加密 (AES)

```
同一个密钥加密和解密:

  明文 + 密钥 → 加密 → 密文
  密文 + 密钥 → 解密 → 明文

AES:
├── 密钥长度: 128/192/256 bit
├── 分组大小: 128 bit
└── 模式: ECB/CBC/CTR/GCM

速度: 非常快 (硬件加速)
用途: 数据加密, 通信加密
```

### 3. 非对称加密 (RSA)

```
公钥加密, 私钥解密 (或反过来):

  公钥: 可以公开
  私钥: 必须保密

  加密: 明文 + 公钥 → 密文
  解密: 密文 + 私钥 → 明文

  签名: 消息哈希 + 私钥 → 签名
  验证: 消息哈希 + 公钥 + 签名 → 有效/无效

速度: 慢 (比 AES 慢 1000 倍)
用途: 密钥交换, 数字签名
```
