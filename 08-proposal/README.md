## 市场分析框架

### PEST 分析
- **P**olitical：政策法规（如数据安全法、环保要求）
- **E**conomic：经济环境（消费力、汇率、供应链）
- **S**ocial：社会趋势（老龄化、健康意识、环保）
- **T**echnological：技术趋势（AI、5G、低功耗）

### Porter 五力模型
```
                 新进入者威胁
                      │
                      ▼
供应商议价力 ◄── 行业竞争 ──► 买方议价力
                      ▲
                      │
                 替代品威胁
```
