# 电子电路基础

## 概述

电子电路是所有消费电子产品的物理基础。本文介绍电阻、电容、二极管、三极管等基本元件的原理和使用方法，以及欧姆定律、分压电路等基础电路分析。

完成本文学习后，你将能够：

- 理解基本电路元件的工作原理
- 使用欧姆定律进行电路计算
- 设计简单的分压、滤波、驱动电路
- 使用万用表和示波器进行基础测量

## 核心内容

### 1. 电压、电流、电阻

```
电压 (Voltage, V):
  推动电流流动的"压力"
  单位: 伏特 (V)
  类比: 水压

电流 (Current, I):
  单位时间内流过的电荷量
  单位: 安培 (A)
  类比: 水流量

电阻 (Resistance, R):
  阻碍电流流动的能力
  单位: 欧姆 (Ω)
  类比: 水管粗细
```

### 2. 欧姆定律

```
V = I × R

  V: 电压 (伏特, V)
  I: 电流 (安培, A)
  R: 电阻 (欧姆, Ω)

变形公式:
  I = V / R
  R = V / I

功率:
  P = V × I = I² × R = V² / R

  P: 功率 (瓦特, W)
```

### 3. 电阻

```
串联: R_total = R1 + R2 + R3
  ──[R1]──[R2]──[R3]──

并联: 1/R_total = 1/R1 + 1/R2 + 1/R3
  ──┬──[R1]──┬──
    │        │
    └──[R2]──┘

色环读数:
  棕(1) 红(2) 橙(3) 黄(4) 绿(5)
  蓝(6) 紫(7) 灰(8) 白(9) 黑(0)

  四环电阻: 前两环=有效数字, 第三环=倍率, 第四环=精度
  五环电阻: 前三环=有效数字, 第四环=倍率, 第五环=精度

常用电阻值:
  100Ω, 220Ω, 330Ω, 470Ω, 1KΩ, 2.2KΩ, 4.7KΩ, 10KΩ, 47KΩ, 100KΩ
```

### 4. 电容

```
储能元件: Q = C × V

  Q: 电荷 (库仑, C)
  C: 电容 (法拉, F)
  V: 电压 (伏特, V)

常用单位:
  1μF (微法) = 10⁻⁶ F
  1nF (纳法) = 10⁻⁹ F
  1pF (皮法) = 10⁻¹² F

常见类型:
  陶瓷电容: 1pF-10μF, 无极性, 高频特性好
  电解电容: 1μF-10000μF, 有极性, 容量大
  钽电容:   0.1μF-1000μF, 有极性, 稳定

用途:
  滤波: 平滑电源波动
  去耦: 消除高频噪声 (100nF 靠近 IC 电源引脚)
  耦合: 传递交流信号, 隔绝直流
  定时: RC 时间常数 τ = R × C
```

### 5. 二极管

```
单向导通:
  正向: 电流从阳极→阴极, 压降 0.7V (硅) 或 0.3V (锗/Schottky)
  反向: 截止, 不导通

LED (发光二极管):
  正向压降: 红 1.8V, 绿 2.2V, 蓝 3.2V, 白 3.0V
  限流电阻: R = (Vcc - Vf) / I_LED

  例: 5V 驱动红色 LED, 10mA
  R = (5 - 1.8) / 0.01 = 320Ω → 选 330Ω

稳压二极管 (Zener):
  反向击穿后保持稳定电压
  用于电压基准和过压保护
```

### 6. 三极管

```
NPN 三极管:
  B (基极): 控制端
  C (集电极): 输出端
  E (发射极): 接地端

  工作条件: Vbe > 0.7V, Ib > 0
  放大: Ic = β × Ib (β = 100-300)
  开关: Ib 足够大时, C-E 导通

MOSFET:
  G (栅极): 控制端 (电压控制)
  D (漏极): 输出端
  S (源极): 接地端

  N-MOS: Vgs > 阈值电压时导通
  P-MOS: Vgs < 阈值电压时导通

  MOSFET 优点: 输入阻抗极高, 无栅极电流
```

## 实践示例

### 示例1：LED 驱动电路

```c
// 计算 LED 限流电阻
// 条件: Vcc=3.3V, LED 红色 Vf=1.8V, 目标电流 I=10mA

// R = (Vcc - Vf) / I
// R = (3.3 - 1.8) / 0.01 = 150Ω

// 选择标准值: 150Ω 或 220Ω (更保守)
// 实际电流: I = (3.3 - 1.8) / 220 = 6.8mA (LED 仍然亮)

// 电路:
// Vcc(3.3V) ──[220Ω]──LED(+)──GND
```

**代码说明**：
- LED 必须串联限流电阻，否则会烧毁
- 电阻值可以根据需要的亮度调整
- 电流越小 LED 越暗，但寿命更长

### 示例2：分压电路

```c
// 分压电路: 将 5V 降到 3.3V
// Vout = Vin × R2 / (R1 + R2)
// 3.3 = 5 × R2 / (R1 + R2)
// R1/R2 = (5-3.3)/3.3 = 0.515

// 选择: R1=10KΩ, R2=19.6KΩ → Vout=3.32V
// 或:   R1=5.1KΩ, R2=10KΩ → Vout=3.31V

// 电路:
// Vin(5V) ──[R1 10K]──┬── Vout(3.3V)
//                      │
//                     [R2 20K]
//                      │
//                     GND
```

**代码说明**：
- 分压电路用于电平转换（如 5V 传感器连接 3.3V MCU）
- 负载电阻应远大于 R2（至少 10 倍），否则输出电压会下降
- 不适合大电流场合，只用于信号电平

## 深入理解

### 原理分析

电容在直流电路中相当于开路（稳态后），在交流电路中阻抗为：
```
Xc = 1 / (2π × f × C)

f: 频率 (Hz)
C: 电容 (F)

频率越高，阻抗越小 → 高频信号更容易通过
→ 这就是为什么电容可以滤波
```

### 最佳实践

1. 电源去耦：每个 IC 的 VCC 引脚放 100nF 陶瓷电容
2. 电解电容注意极性，接反会爆炸
3. 电阻功率留 2 倍余量（实际功耗 0.1W 就选 0.25W 的）
4. LED 不要忘记限流电阻
5. 高频电路用小封装（0402/0603），低频用大封装（0805/1206）

## 常见问题

### Q1: 为什么 LED 必须串联电阻？

**A**: LED 是电流型器件，正向导通后压降基本恒定（约 1.8-3.2V）。如果没有电阻限流，电流会无限增大直到烧毁 LED。串联电阻限制电流在安全范围内（通常 5-20mA）。

### Q2: 电容串联和并联怎么算？

**A**: 电容的串并联与电阻相反：
- 并联：C_total = C1 + C2 + C3（容量相加）
- 串联：1/C_total = 1/C1 + 1/C2 + 1/C3（容量减小）
