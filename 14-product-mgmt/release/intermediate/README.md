### 结构设计
```
消费电子产品结构设计要点:

外壳材料选择:
| 材料 | 特点 | 成本 | 适用 |
|------|------|------|------|
| ABS | 韧性好, 易加工 | 低 | 入门产品 |
| PC | 强度高, 透明 | 中 | 高端产品 |
| PC+ABS | 综合性能好 | 中 | 通用 |
| 铝合金 | 散热好, 质感 | 高 | 旗舰产品 |
| 镁合金 | 轻量, 强度 | 高 | 航拍设备 |
| 碳纤维 | 超轻, 高强度 | 很高 | 竞速/专业 |

结构设计检查清单:
□ 壁厚均匀 (1.5-3mm)
□ 拔模角 ≥ 1°
□ 加强筋设计 (高度 ≤ 3×壁厚)
□ 圆角设计 (避免应力集中)
□ 螺柱设计 (内径=螺丝外径-0.2mm)
□ 卡扣设计 (可反复拆装)
□ 防水设计 (密封圈/超声波焊接)
□ 散热设计 (导热垫/散热筋)
□ 跌落设计 (边角加固)
□ 防滑设计 (橡胶脚垫/纹理)
```

### 2. 热设计

### 热设计流程
```
热设计流程:

需求分析
├── 功耗预算 (总功耗, 各模块功耗)
├── 温度限制 (外壳 ≤ 45°C, 芯片 ≤ 85°C)
└── 环境条件 (最高环境温度)

热仿真
├── 建立热模型 (芯片/PCB/外壳)
├── 仿真工具: FloTHERM / Icepak / FLOTHERM
├── 仿真结果: 温度分布图
└── 迭代优化

散热方案
├── 自然对流 (被动散热)
│   ├── 散热筋 (增加表面积)
│   ├── 导热垫 (芯片到外壳)
│   └── 散热铜皮 (PCB 内层)
│
├── 强制对流 (主动散热)
│   ├── 风扇 (小型/微型)
│   ├── 热管 (均热)
│   └── 均热板 (Vapor Chamber)
│
└── 其他
    ├── 导热硅脂 (芯片到散热器)
    ├── 相变材料 (吸热)
    └── TEC (制冷片, 特殊场景)

热设计检查清单:
□ 热源远离热敏元件
□ 散热路径通畅 (无遮挡)
□ 导热材料选型正确
□ 风道设计合理 (进风口→热源→出风口)
□ 温度监测点设置
□ 降功率保护阈值设定
□ 最高环境温度测试通过
```

### 常见散热方案
```
典型产品散热设计:

无人机飞控:
├── 功耗: 1-3W
├── 方案: PCB 铜皮散热 + 导热垫到外壳
└── 注意: 远离电机热源

手持云台:
├── 功耗: 2-5W
├── 方案: 铝合金外壳散热 + 热管均热
└── 注意: 手持温度 ≤ 42°C

扫地机器人:
├── 功耗: 30-60W (含风机)
├── 方案: 风机散热 + 散热筋
└── 注意: 尘盒堵塞影响散热

3D 打印机:
├── 热端: 260°C (隔热设计)
├── 热床: 100°C (保温设计)
├── 驱动: TMC 芯片 (散热垫)
└── 注意: 热端与冷端隔热
```
