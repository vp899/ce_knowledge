# 打板制造流程

## 概述

本文介绍 manufacturing 领域的 beginner 级别知识。

完成本文学习后，你将能够：

- 理解 SMT 贴片工艺流程
- 掌握 BOM 管理和元器件选型
- 了解 PCBA 测试方法

## 背景知识

### 相关概念

### 前置知识

- C 语言基础 (变量/指针/函数)
- 基本电子电路知识 (电压/电流/电阻)
- 基本数学 (三角函数/向量)

## 核心内容

### 1. [[pcb-layout|PCB]] 制造流程

### 制造工艺流程
```
Gerber 文件输出
    │
    ▼
工程审核 (DFM 检查)
    │
    ▼
开料 (Cutting)
    │ ├── 内层图形转移 (Inner Layer Imaging)
    │ ├── 内层蚀刻 (Inner Layer Etch)
    │ └── 内层 AOI 检查
    ▼
层压 (Lamination)
    │ ├── 棕化 (Brown Oxide)
    │ ├── 铆合 (Riveting)
    │ └── 压合 (Pressing)
    ▼
钻孔 (Drilling)
    │ ├── 机械钻孔
    │ └── 激光钻孔 (HDI)
    ▼
沉铜 (Electroless Copper)
    │
    ▼
外层图形转移 (Outer Layer Imaging)
    │
    ▼
电镀 (Electroplating)
    │ ├── 图形电镀
    │ └── 电镀锡
    ▼
外层蚀刻 (Outer Layer Etch)
    │
    ▼
阻焊 (Solder Mask)
    │ ├── 印刷阻焊油墨
    │ ├── 曝光
    │ └── 显影
    ▼
丝印 (Silkscreen)
    │
    ▼
表面处理 (Surface Finish)
    │ ├── HASL (喷锡)
    │ ├── ENIG (沉金)
    │ ├── OSP
    │ └── 沉银
    ▼
成型 (Routing)
    │
    ▼
电测 (Electrical Test)
    │
    ▼
终检 (Final Inspection)
    │
    ▼
包装出货
```

### DFM (可制造性设计) 检查
```
线宽/线距:
├── 最小线宽: 4mil (0.1mm) 常规, 3mil (0.075mm) 高端
├── 最小线距: 4mil (0.1mm) 常规, 3mil (0.075mm) 高端
└── 铜厚: 1oz (35μm) 常规, 2oz (70μm) 电源

过孔:
├── 最小孔径: 0.2mm 常规, 0.1mm HDI
├── 最小焊盘: 孔径 + 0.25mm
├── 过孔间距: ≥ 0.5mm
└── 盲埋孔: HDI 设计

阻焊:
├── 最小阻焊桥: 0.1mm
├── 阻焊开窗: 焊盘 + 0.05mm
└── BGA 阻焊: 盘中盘/非阻焊定义

丝印:
├── 最小线宽: 0.15mm
├── 最小字高: 0.8mm
├── 丝印间距: ≥ 0.2mm
└── 不可覆盖焊盘

拼板:
├── 拼板尺寸: 满足设备要求
├── 工艺边: ≥ 5mm
├── 连接方式: V-CUT / 邮票孔
└── 定位孔: 3 个 (不对称)
```

### 3. SMT 贴片

### SMT 工艺流程
```
PCB 上板
    │
    ▼
锡膏印刷 (Solder Paste Printing)
    │ ├── 钢网设计
    │ ├── 锡膏选择
    │ └── 印刷参数
    ▼
SPI 检测 (Solder Paste Inspection)
    │
    ▼
贴片 (Pick & Place)
    │ ├── 高速贴片机 (小元件)
    │ └── 多功能贴片机 (大元件)
    ▼
回流焊 (Reflow Soldering)
    │ ├── 预热区 (150°C, 60-90s)
    │ ├── 恒温区 (180°C, 60-90s)
    │ ├── 回流区 (230-250°C, 30-60s)
    │ └── 冷却区
    ▼
AOI 检测 (Automated Optical Inspection)
    │
    ▼
X-Ray 检测 (BGA 等隐藏焊点)
    │
    ▼
功能测试
    │
    ▼
下板
```

### 锡膏印刷参数
```
钢网:
├── 材质: 不锈钢
├── 厚度: 0.1-0.15mm (常规), 0.08mm (精细间距)
├── 开口: 按焊盘尺寸设计
└── 表面处理: 电抛光 / 纳米涂层

锡膏:
├── 合金: SAC305 (Sn96.5/Ag3.0/Cu0.5)
├── 粉末类型: Type 4 (20-38μm) / Type 5 (15-25μm)
├── 助焊剂: 免清洗 / 水溶性
└── 保存: 2-10°C, 回温 4h

印刷参数:
├── 压力: 0.5-2.0 kg
├── 速度: 20-80 mm/s
├── 脱模距离: 0.5-2mm
└── 脱模速度: 0.5-2 mm/s
```

### 4. 组装与测试

### 组装流程
```
SMT 贴片完成
    │
    ▼
DIP 插件 (如有)
    │ ├── 手工插件
    │ └── 波峰焊
    ▼
组装
    │ ├── 外壳组装
    │ ├── 连接器安装
    │ ├── 线缆连接
    │ └── 标签粘贴
    ▼
功能测试
    │ ├── ICT (在线测试)
    │ ├── FCT (功能测试)
    │ └── 老化测试
    ▼
外观检查
    │
    ▼
包装
    │ ├── 产品包装
    │ ├── 附件放入
    │ └── 外箱封装
    ▼
OQC (出货检验)
    │
    ▼
入库/出货
```

### 测试治具设计
```
ICT 治具:
├── 探针定位
├── 测试点覆盖
├── 电源接入
└── 信号测量

FCT 治具:
├── 功能接口
├── 模拟负载
├── 自动化脚本
└── 判定标准

治具设计检查清单:
□ 测试点覆盖率 > 95%
□ 探针定位精度 ±0.05mm
□ 夹具压力适中
□ 测试时间 < 30s
□ 误判率 < 0.1%
□ 漏判率 < 0.01%
```

### 5. 品质管理

### 品质检验标准
```
IQC (来料检验):
├── 外观检查
├── 尺寸测量
├── 电气测试
├── 可焊性测试
└── 抽样标准: AQL 0.65

IPQC (过程检验):
├── 首件检验
├── 巡回检验
├── 锡膏检测
├── 回流焊温度曲线
└── AOI 检测

OQC (出货检验):
├── 外观检查
├── 功能测试
├── 包装检查
├── 标签检查
└── 抽样标准: AQL 1.0
```

### 良率分析
```
良率计算:
良率 = (投入数 - 不良数) / 投入数 × 100%

不良分析:
├── 锡膏不良 (少锡/多锡/偏移)
├── 贴片不良 (偏移/立碑/翻件)
├── 焊接不良 (虚焊/桥接/冷焊)
├── 器件不良 (损坏/错件/方向)
└── 组装不良 (划伤/异物/装配)

改进措施:
├── 优化钢网设计
├── 调整回流焊曲线
├── 改善贴片程序
├── 加强来料检验
└── 完善作业规范
```

### 6. 成本控制

### 成本结构分析
```
BOM 成本:
├── 主芯片: 30-40%
├── 被动元件: 10-15%
├── 连接器: 5-10%
├── 结构件: 10-20%
├── 电池: 5-15%
└── 其他: 5-10%

制造成本:
├── SMT 贴片: 0.01-0.05 元/点
├── DIP 插件: 0.05-0.1 元/点
├── 组装: 1-5 元/台
├── 测试: 0.5-2 元/台
└── 包装: 1-3 元/台

其他成本:
├── 模具分摊
├── 认证分摊
├── 研发分摊
├── 物流成本
└── 售后成本
```

### Q1: 入门需要哪些前置知识？

**A**: 需要基础的 C 语言编程能力和基本的电子电路知识。建议先完成前置模块的学习。
