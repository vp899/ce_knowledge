# PCB Layout 设计

## 1. 叠层设计

### 2 层板
```
Layer 1 (Top):    信号 + 元件
Layer 2 (Bottom): 信号 + 地平面

适用: 简单低速设计、成本敏感
特点: 无完整地平面，EMC 性能较差
```

### 4 层板 (推荐)
```
Layer 1 (Top):       信号 + 元件
Layer 2 (Inner 1):   地平面 (GND) ★ 完整地平面
Layer 3 (Inner 2):   电源平面 (VCC)
Layer 4 (Bottom):    信号 + 元件

特点:
- 信号层紧邻地平面，回流路径短
- 电源和地平面形成平板电容，利于去耦
- 成本适中，适合大多数消费电子
```

### 6 层板
```
Layer 1 (Top):       高速信号 + 元件
Layer 2 (Inner 1):   地平面 (GND)
Layer 3 (Inner 2):   信号 (低速)
Layer 4 (Inner 3):   电源平面 (VCC)
Layer 5 (Inner 4):   地平面 (GND)
Layer 6 (Bottom):    信号 + 元件

特点:
- 两个完整地平面，EMC 性能好
- 高速信号在顶层，紧邻地平面
- 适合高速设计 (DDR, USB3.0, LVDS)
```

### 阻抗控制
```
单端阻抗 (微带线):
Z0 = 87 / sqrt(εr + 1.41) * ln(5.98*h / (0.8*w + t))
- εr: 介电常数 (FR4 ≈ 4.2-4.5)
- h: 介质厚度
- w: 线宽
- t: 铜厚

典型阻抗要求:
- 通用数字信号: 50Ω ± 10%
- USB 2.0: 90Ω 差分
- LVDS: 100Ω 差分
- 以太网: 100Ω 差分
- RF 50Ω: 50Ω ± 5%

常用 FR4 (εr=4.3) 阻抗参考:
| 叠层 | 线宽 | 阻抗 |
|------|------|------|
| 4层, 0.2mm介质 | 0.15mm | 50Ω |
| 4层, 0.2mm介质 | 0.10mm | 70Ω |
| 6层, 0.1mm介质 | 0.12mm | 50Ω |
```

## 2. 布局规则

### 元件布局优先级
```
1. 固定元件 (连接器、按键、显示屏)
2. 大元件 (IC、模块)
3. 电源元件 (DC-DC、LDO、电感)
4. 去耦电容 (靠近 IC 电源引脚)
5. 小元件 (电阻、电容)
```

### 布局检查清单
```
□ 连接器位置符合外壳结构
□ 按键/LED 位置与外壳对齐
□ 发热元件远离热敏元件
□ 晶振靠近 MCU
□ 去耦电容靠近 IC 电源引脚
□ 天线区域净空 (无铜、无元件)
□ 测试点可及
□ 丝印清晰可读
```

## 3. 布线规则

### 线宽与电流
| 线宽 (外层) | 线宽 (内层) | 电流 (1oz 铜) | 电流 (2oz 铜) |
|-------------|-------------|---------------|---------------|
| 0.15mm | 0.20mm | 0.3A | 0.5A |
| 0.20mm | 0.30mm | 0.5A | 0.8A |
| 0.30mm | 0.50mm | 0.8A | 1.2A |
| 0.50mm | 0.80mm | 1.2A | 1.8A |
| 1.00mm | 1.50mm | 2.0A | 3.0A |
| 2.00mm | 3.00mm | 3.5A | 5.0A |

### 走线规则
```
规则 1: 3W 规则
    走线间距 ≥ 3 倍线宽
    减少串扰 (70% 串扰消除)

规则 2: 20H 规则
    电源平面边缘缩进 ≥ 20 倍介质厚度
    减少边缘辐射

规则 3: 关键信号走线
    - 时钟信号: 走内层，紧邻地平面
    - 高速信号: 控制阻抗，减少过孔
    - 差分对: 等长、等距、对称
    - 模拟信号: 远离数字信号，单独地回路

规则 4: 过孔使用
    - 信号过孔: 0.3mm 孔径 / 0.5mm 焊盘
    - 电源过孔: 0.5mm 孔径 / 0.8mm 焊盘
    - 过孔间距: ≥ 0.5mm
    - 高速信号: 减少过孔数量
```

```
│                                       │

│                                       │
│  线宽: 0.12mm                         │

```

## 4. 电源平面设计

### 电源分割
```
┌─────────────────────────────────────────────┐
│                 内层电源平面                   │
│                                               │
│  ┌─────────────────┐  ┌─────────────────┐    │
│  │                  │  │                  │    │
│  │   3.3V 区域      │  │   1.8V 区域      │    │
│  │                  │  │                  │    │
│  │                  │  │                  │    │
│  └─────────────────┘  └─────────────────┘    │
│                                               │
│  ┌──────────────────────────────────────┐    │
│  │                                       │    │
│  │   5V 区域                             │    │
│  │                                       │    │
│  └──────────────────────────────────────┘    │
│                                               │
│  分割间距: ≥ 0.5mm                           │
│  跨分割信号: 添加缝合过孔                     │
└─────────────────────────────────────────────┘
```

### 地平面完整性
```
关键原则:
1. 地平面尽量完整，避免大面积开槽
2. 信号换层时，在过孔附近放置地过孔
3. 高速信号下方地平面不能有分割
4. 模拟地和数字地单点连接
5. 大面积铺铜增加散热和屏蔽
```

## 5. EMC 设计

### EMC 设计检查清单
```
□ 所有 I/O 接口有 ESD 保护
□ 电源入口有 EMI 滤波
□ 时钟信号有阻尼电阻
□ 高速信号走线在地平面上方
□ 电源/地平面对完整
□ 连接器地与板地良好连接
□ 天线区域净空
□ 金属外壳多点接地
□ 走线避免直角 (用 45° 或圆弧)
□ 关键信号包地处理
```

### EMI 滤波设计
```
电源入口 EMI 滤波:

VCC_IN ── [L_CM] ──┬── [L_d] ── VCC_OUT
                    │
                   [C_x]
                    │
                   GND

L_CM: 共模电感 (抑制共模噪声)
L_d: 差模电感 (抑制差模噪声)
C_x: X 电容 (0.1-1μF, 跨差模)
C_y: Y 电容 (2.2-4.7nF, 到地)

信号线 EMI 滤波:
信号 ── [R 100Ω] ──┬── 输出
                    │
                   [C 100pF]
                    │
                   GND
```

## 6. 散热设计

### 热设计规则
```
1. 热源 (DC-DC, MCU, 功率管) 远离热敏元件
2. 大面积铜皮散热 (连接到 IC 热焊盘)
3. 散热过孔 (thermal via):
   - 直径: 0.3mm
   - 间距: 1.0mm
   - 数量: 3x3 或更多
   - 连接到内层地平面
4. 必要时添加散热片或导热垫
5. 金属外壳可作为散热器
```

### 功率元件散热
```
DC-DC IC 散热:
┌────────────────────────┐
│         IC 封装          │
│  ┌──────────────────┐  │
│  │    Exposed Pad    │  │
│  │    (热焊盘)        │  │
│  └──────────────────┘  │
│          │││            │
│        散热过孔          │
│          │││            │
│     内层大面积铜皮       │
└────────────────────────┘
```

## 7. 丝印设计

### 丝印规范
```
1. 元件标号: 位置靠近元件，方向一致
2. 极性标记: 二极管、电容、IC Pin1
3. 测试点标注: TP1, TP2, 信号名
4. 连接器引脚标注: Pin1, VCC, GND
5. 版本号、日期、公司 Logo
6. 丝印不覆盖焊盘
7. 丝印线宽 ≥ 0.15mm
8. 丝印字高 ≥ 0.8mm
```
