## 核心内容

### 1. 基本元件

### 电阻
```
欧姆定律: V = I × R

  V: 电压 (伏特, V)
  I: 电流 (安培, A)
  R: 电阻 (欧姆, Ω)

串联: R_total = R1 + R2 + R3
并联: 1/R_total = 1/R1 + 1/R2 + 1/R3

功率: P = V × I = I² × R = V² / R

色环读数:
  棕(1) 红(2) 橙(3) 黄(4) 绿(5)
  蓝(6) 紫(7) 灰(8) 白(9) 黑(0)
```

### 电容
```
储能元件: Q = C × V

  Q: 电荷 (库仑, C)
  C: 电容 (法拉, F)
  V: 电压 (伏特, V)

常用单位:
  1μF = 10⁻⁶ F
  1nF = 10⁻⁹ F
  1pF = 10⁻¹² F

用途:
├── 滤波: 平滑电源波动
├── 去耦: 消除高频噪声
├── 耦合: 传递交流信号
└── 定时: RC 时间常数 τ = R × C
```

### 二极管
```
单向导通:
  正向: 电流从阳极→阴极, 压降 0.3V(Schottky) 或 0.7V(Si)
  反向: 截止, 不导通

LED (发光二极管):
  正向压降: 红 1.8V, 绿 2.2V, 蓝 3.2V
  限流电阻: R = (Vcc - Vf) / I
  例: 5V 红色 LED, 10mA → R = (5-1.8)/0.01 = 320Ω
```

### 2. 基本电路

### 分压电路
```
Vout = Vin × R2 / (R1 + R2)

  Vin ──[R1]──┬── Vout
               │
              [R2]
               │
              GND

例: 5V → 3.3V
  3.3 = 5 × R2/(R1+R2)
  R1/R2 = 0.515
  选 R1=10K, R2=20K → Vout=3.33V
```

### LED 驱动
```
Vcc ──[R]──LED──GND

R = (Vcc - Vf) / I_LED

例: 3.3V 驱动红色 LED (Vf=1.8V, I=10mA)
R = (3.3-1.8)/0.01 = 150Ω
```
