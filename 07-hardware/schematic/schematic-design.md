## 核心内容

### 1. 电源系统设计

### 电源拓扑设计
```
电池 (3.7V 锂电池)
    │
    ├── 充电管理 IC (TP4056 / BQ24075)
    │       │
    │       ├── USB 输入 (5V)
    │       ├── 充电电流设置 (R_PROG)
    │       ├── 充电状态指示
    │       └── 电池保护 (过充/过放/过流)
    │
    ├── DC-DC 升压 (TPS61088) → 5V rail
    │       │
    │       ├── USB 供电
    │       ├── 电机驱动
    │       └── 显示屏
    │
    ├── DC-DC 降压 (TPS63020) → 3.3V rail
    │       │
    │       ├── MCU (STM32)
    │       ├── 传感器
    │       ├── Flash / EEPROM
    │       └── 通信模块 (BLE/WiFi)
    │
    └── LDO (AMS1117-1.8) → 1.8V rail
            │
            ├── 内核供电
            └── 部分 IO 供电
```

### 电源设计要点

#### DC-DC 选型参数
| 参数 | 说明 | 典型值 |
|------|------|--------|
| 输入电压范围 | 覆盖电池电压波动 | 2.7V - 4.5V |
| 输出电压 | 满足系统需求 | 3.3V ± 2% |
| 输出电流 | 满足峰值需求 + 余量 | 1.5A (实际需求 1A) |
| 效率 | 影响续航 | >90% @ 负载 |
| 静态电流 | 影响待机功耗 | <50μA |
| 开关频率 | 影响效率和 EMI | 1-2MHz |
| 封装 | 散热考虑 | QFN / SOT-23 |

#### 电源时序设计
```
Power On Sequence:
    
t=0     VBAT 上升
t=1ms   LDO_1.8V 稳定 (内核先上电)
t=5ms   DC-DC_3.3V 稳定 (IO 供电)
t=10ms  MCU 复位释放
t=20ms  MCU 开始执行
t=50ms  外设供电使能 (EN 引脚)
t=100ms 系统就绪

Power Off Sequence:
    
t=0     关机请求
t=10ms  保存数据到 Flash
t=20ms  关闭外设
t=30ms  MCU 进入待机
t=50ms  DC-DC 关断
```

#### 去耦电容设计规则
```
规则 1: 每个电源引脚放置 100nF 陶瓷电容 (X7R, 0402)
规则 2: 每组电源放置 10μF 电解/陶瓷电容 (bulk cap)
规则 3: 高速芯片 (MCU, FPGA) 放置 1μF + 100nF + 10nF
规则 4: 去耦电容靠近引脚放置 (< 3mm)
规则 5: 高频电容直接连接到电源/地引脚，不经过过孔
```

### 2. 时钟系统

### 晶振设计
```c
/* 外部高速晶振 (HSE) - 8MHz */
/* 负载电容计算: CL = (C1 * C2) / (C1 + C2) + C_stray
 * 典型: CL = 20pF, C_stray = 5pF
 * C1 = C2 = 2 * (CL - C_stray) = 30pF
 */

/* 外部低速晶振 (LSE) - 32.768kHz */
/* 用于 RTC 和低功耗定时
 * 负载电容: CL = 12.5pF
 * C1 = C2 = 2 * (12.5 - 3) = 19pF → 选 18pF 或 20pF
 */
```

### 晶振布局要点
```
┌─────────────────────────────────────────┐
│  MCU                                      │
│     OSC_IN ──┬──[R_feedback]──┬── OSC_OUT│
│              │                │           │
│             C1               C2           │
│              │                │           │
│             GND              GND          │
│                                           │
│  布局规则:                                │
│  1. 晶振靠近 MCU (距离 < 5mm)            │
│  2. 走线尽量短且等长                      │
│  3. 晶振下方铺地铜，打过孔到内层地        │
│  4. 晶振周围用地线包围 (guard ring)       │
│  5. 避免高速信号线从晶振区域穿过          │
└─────────────────────────────────────────┘
```

### 3. 复位电路

### 复位电路设计
```
               VCC (3.3V)
                │
               [R1] 10K
                │
MCU_NRST ───────┤
                │
               [C1] 100nF
                │
               GND

复位时间: t = R1 * C1 = 10K * 100nF = 1ms

可选: 添加手动复位按键
MCU_NRST ─── [S1] ─── GND
```

### 看门狗复位
```
MCU ─── IWDG ──── 超时复位
         │
         ├── 喂狗周期: < 超时时间
         └── 超时时间: 由应用需求决定

外部看门狗 (TPS3823):
WDI ──── MCU GPIO (喂狗信号)
RESET ── MCU NRST
         │
         └── 超时: 1.6s (典型)
```

### 4. 调试接口

### SWD 接口定义
```
┌─────────────────────────┐
│  10-pin SWD Connector    │
│                          │
│  1: VCC (3.3V)           │
│  2: SWDIO                │
│  3: GND                  │
│  4: SWCLK                │
│  5: GND                  │
│  6: SWO (可选)           │
│  7: NC                   │
│  8: NC                   │
│  9: GND                  │
│  10: NRST (可选)         │
└─────────────────────────┘

连接器选型:
- 2.54mm 排针 (开发阶段)
- 1.27mm 迷你连接器 (量产)
- Tag-Connect (无连接器，量产)
```

### UART 调试串口
```
MCU_TX (PA9)  ──── 1K ──── TXD (USB-UART 转换器)
MCU_RX (PA10) ──── 1K ──── RXD

添加限流电阻保护 MCU 引脚
可选: 添加 ESD 保护二极管
```

### 5. ESD 保护设计

### 接口 ESD 保护方案
```
USB 接口:
VCC ── [TVS 5.5V] ── GND
D+  ── [TVS 3.3V] ── GND
D-  ── [TVS 3.3V] ── GND

按键输入:
GPIO ── [R 100R] ── [TVS 3.3V] ── GND
         │
        [C 100pF]
         │
        GND

传感器接口 (I2C):
SCL ── [R 100R] ── [TVS 3.3V] ── GND
SDA ── [R 100R] ── [TVS 3.3V] ── GND

天线接口 (BLE/WiFi):
ANT ── [ESD 保护器] ── GND
         │
         └── 选择低电容 ESD 器件 (<1pF)
```

### 6. 原理图检查清单

### 电气规则检查
- [ ] 所有电源引脚已连接
- [ ] 电源极性正确
- [ ] 去耦电容数量足够
- [ ] 上拉/下拉电阻值合理
- [ ] 限流电阻计算正确
- [ ] ESD 保护器件放置合理
- [ ] 未使用引脚处理 (上拉/下拉/悬空)
- [ ] 电源指示 LED 有限流电阻

### 信号完整性检查
- [ ] 高速信号阻抗匹配
- [ ] 差分对走线等长
- [ ] 时钟信号有串联阻尼电阻
- [ ] 中断信号有上拉电阻
- [ ] ADC 参考电压稳定
- [ ] 模拟/数字地分割正确

### 可靠性检查
- [ ] 电源反接保护 (P-MOS / 二极管)
- [ ] 过压保护 (TVS / 稳压管)
- [ ] 过流保护 (保险丝 / 电子保险)
- [ ] 热设计 (散热焊盘、过孔)
- [ ] 关键信号有测试点
- [ ] 预留调试接口

### 7. 常用电路模块

### LED 驱动电路
```
        VCC (3.3V)
         │
        [R] 限流电阻
         │
        LED (正极)
         │
        LED (负极)
         │
        GPIO (MCU)

限流电阻计算:
R = (VCC - V_forward) / I_LED
例: R = (3.3V - 2.0V) / 10mA = 130Ω → 选 150Ω
```

### MOSFET 开关电路
```
                    VCC
                     │
                   [负载]
                     │
                   MOSFET (S)
                     │
                   MOSFET (D)
                     │
                    GND

栅极驱动:
GPIO ── [R_gate 100R] ── MOSFET (G)
                              │
                            [R_pullup 100K]
                              │
                              VCC (或 GND，取决于默认状态)
```

### 运放信号调理
```
传感器输出 ── [R1] ──┬── [-] OPA ── 输出
                     │       │
                    [R2]    [R_f]
                     │       │
                    GND      ├─── OPA (+)
                             │
                           [R_g]
                             │
                           V_ref

增益: G = -R_f / R1 (反相放大)
偏置: V_out = V_ref * (1 + R_f/R1) - V_in * (R_f/R1)
```
