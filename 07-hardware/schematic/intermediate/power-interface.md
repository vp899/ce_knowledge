## 核心内容

### 1. LDO 线性稳压器

```
原理: 通过可变电阻消耗多余电压

  Vin ──[调整管]── Vout
                │
               [反馈]
                │
               GND

Vout = Vref × (1 + R1/R2)

优点: 低噪声, 简单
缺点: 效率低 = Vout/Vin

例: 5V → 3.3V, 100mA
  效率 = 3.3/5 = 66%
  功耗 = (5-3.3)×0.1 = 0.17W

适用: 小电流, 噪声敏感 (模拟电路, PLL)
```

### 2. DC-DC 开关稳压器

```
Buck (降压):
  Vin ──[开关]──[电感]── Vout
                  │
                 [电容]
                  │
                 GND

  Vout = Vin × D (D = 占空比)
  效率: 85-95%

Boost (升压):
  Vin ──[电感]──[开关]── Vout
                 │
                [二极管]
                 │
                [电容]

  Vout = Vin / (1-D)

Buck-Boost (升降压):
  可以输出比输入高或低的电压
```

### 3. 常用接口电路

### UART 电平转换
```
3.3V MCU ←→ 5V 设备:

  3.3V_TX ──[R 1K]── 5V_RX (直接连接, 5V 兼容 3.3V)

  5V_TX ──[分压]── 3.3V_RX
         ├──[R1 2K]──┤
         │            │
         └──[R2 3K]──GND
         输出: 5V × 3K/(2K+3K) = 3V ✓
```

### I2C 上拉
```
I2C 需要上拉电阻:

  SCL ──[R 4.7K]── Vcc (3.3V)
  SDA ──[R 4.7K]── Vcc (3.3V)

选择依据:
  R_min = (Vcc - 0.4V) / 3mA
  R_max = 限制上升时间
  
  400kHz: 1.5K-4.7K
  100kHz: 4.7K-10K
```

## 常见问题

### Q1: 如何调试复杂问题？

**A**: 使用逻辑分析仪/示波器抓取信号；添加日志输出关键变量；使用 GDB 在线调试；分模块隔离问题。

### Q2: 性能不够怎么办？

**A**: 使用 DMA 减少 CPU 负担；优化中断处理 (Top/Bottom Half)；使用硬件加速器；降低采样率或简化算法。
