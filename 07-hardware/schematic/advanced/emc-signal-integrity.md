## 核心内容

### 1. EMC 设计

### 辐射发射抑制
```
辐射来源:
├── 差模辐射: 电流环路
└── 共模辐射: 线缆天线效应

抑制措施:
├── 滤波: 共模电感 + X/Y 电容
├── 屏蔽: 金属外壳 + 导电泡棉
├── 接地: 多点接地, 地平面完整
└── 布局: 高速信号远离缝隙

共模滤波器:
  ┌──[Lcm]──┐
  │         │
  Line ──┤         ├── Line
  │         │
  └──[Lcm]──┘
```

### ESD 防护
```
TVS 二极管选型:
├── 工作电压 > 信号电压
├── 钳位电压 < 芯片耐受
├── 响应时间 < 1ns
└── 电容 < 1pF (高速信号)

放置位置:
  尽量靠近连接器

  Connector ──[TVS]──[R 100Ω]── MCU
```

### 2. 信号完整性

### 阻抗匹配
```
传输线阻抗:
  Z0 = sqrt(L/C)

单端 50Ω:
  微带线: w/h ≈ 2 (FR4)

差分 100Ω:
  线间距 = 线宽

反射系数:
  Γ = (ZL - Z0) / (ZL + Z0)

终端匹配:
  串联: R + Z0 = ZL (源端)
  并联: R = ZL (终端)
```

### 电源完整性
```
去耦策略:
├── 10μF: 板级 bulk
├── 1μF: 芯片供电
├── 100nF: 高频去耦
└── 10nF: 超高频

放置原则:
  100nF 尽量靠近电源引脚
  走线尽量短
  直接连接到地平面

电源平面:
  电源和地平面形成平板电容
  → 天然的高频去耦
```
