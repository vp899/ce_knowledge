## 核心内容

### 1. 角点检测

### Harris 角点
```
角点: 两个方向梯度都大的点

M = [Ix²  IxIy]
    [IxIy Iy² ]

角点响应: R = det(M) - k·trace(M)²
  det(M) = Ix²·Iy² - (IxIy)²
  trace(M) = Ix² + Iy²

  R > 阈值 → 角点
  R < 0 → 边缘
  |R| 小 → 平坦区域
```

### FAST 角点 (更快)
```
以像素 p 为中心, 半径 3 的圆上 16 个像素

判断条件:
  连续 N 个 (通常 N=12) 像素亮度 > Ip+t 或 < Ip-t
  → p 是角点

加速: 先检查 1, 5, 9, 13 四个点
  至少 3 个满足才继续检测

速度: 比 Harris 快 10-100 倍
```

### 2. 光流法

### Lucas-Kanade 光流
```
假设: 小窗口内运动一致

  Ix·u + Iy·v + It = 0

  u, v = 光流速度 (dx/dt, dy/dt)
  Ix, Iy = 空间梯度
  It = 时间梯度

最小二乘求解:
  [u]   [ΣIx²   ΣIxIy]⁻¹ [ΣIxIt]
  [v] = [ΣIxIy  ΣIy² ]   [ΣIyIt]
```

### 金字塔光流
```
从粗到细, 逐层追踪:

Level 2 (1/4 分辨率): 粗略运动
    ↓
Level 1 (1/2 分辨率): 中等精度
    ↓
Level 0 (原图): 精确位置

优势: 大幅度运动也能追踪
```

### 3. 目标跟踪

### KCF 相关滤波
```
核心思想: 用相关滤波器在下一帧中找到目标

  训练: 在目标区域训练滤波器
  检测: 在搜索区域计算响应图
  更新: 用新位置更新滤波器

优点: 速度极快 (>100fps)
缺点: 对遮挡敏感, 尺度固定
```
