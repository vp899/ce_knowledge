## 核心技术详解

### 1. LDS 激光雷达 (扫地机专用)

```
扫地机 LDS LiDAR 特点:
├── 测距原理: 三角测距 (低成本) 或 ToF (高精度)
├── 扫描方式: 电机旋转 (300-600 RPM)
├── 测距范围: 0.1-12m (典型)
├── 角分辨率: 0.5-1°
├── 扫描频率: 5-15Hz
├── 精度: ±1-3cm
├── 采样率: 2000-8000 点/秒
└── 成本: ¥30-150

典型 LDS 模块:
| 型号 | 厂商 | 原理 | 测距 | 精度 | 成本 |
|------|------|------|------|------|------|
| YDLIDAR X4 | EAI | 三角 | 0.12-10m | ±1% | ¥50 |
| RPLIDAR A1 | SLAMTEC | 三角 | 0.15-12m | ±1% | ¥80 |
| LDS-02 | 三星 | ToF | 0.1-8m | ±2% | ¥100 |
| 定制模块 | 各厂 | ToF | 0.05-10m | ±1.5% | ¥40 |

LDS 数据输出格式:
角度(°), 距离(mm), 强度, 起始标志
0.0, 1234, 156, 1
0.5, 1236, 154, 0
1.0, 1240, 152, 0
...
359.5, 1230, 158, 0
```

### 3. 路径规划算法

```
全覆盖清扫算法:

弓字形清扫 (Zigzag):
┌─────────────────────────────┐
│ →→→→→→→→→→→→→→→→→→→→→→→→ │
│ ←←←←←←←←←←←←←←←←←←←←←←← │
│ →→→→→→→→→→→→→→→→→→→→→→→→ │
│ ←←←←←←←←←←←←←←←←←←←←←←← │
│ →→→→→→→→→→→→→→→→→→→→→→→→ │
└─────────────────────────────┘
优点: 覆盖率高, 实现简单
缺点: 效率低, 大量重复

分区清扫:
┌─────────┬─────────┐
│ 区域 A  │ 区域 B  │
│ →→→→→→ │ →→→→→→ │
│ ←←←←←← │ ←←←←←← │
├─────────┼─────────┤
│ 区域 C  │ 区域 D  │
│ →→→→→→ │ →→→→→→ │
│ ←←←←←← │ ←←←←←← │
└─────────┴─────────┘
优点: 有序, 可暂停恢复
缺点: 分区算法复杂

路径规划算法:
├── Boustrophedon (牛耕式): 基础全覆盖
├── Spanning Tree: 基于地图的最优路径
├── Neural Network: 学习型规划
└── A* + 覆盖约束: 最短路径 + 全覆盖
```

### 4. 避障系统

```
扫地机避障技术演进:

第 1 代: 碰撞 + 红外
├── 前碰撞板: 机械开关
├── 沿墙红外: 侧面测距 (2-10cm)
├── 悬崖红外: 底部测距 (防止跌落)
└── 优点: 便宜; 缺点: 被动避障

第 2 代: LDS + 红外
├── LDS 建图: 已知障碍物位置
├── 红外辅助: 近距离补充
└── 优点: 主动避障; 缺点: 低矮障碍物盲区

第 3 代: 视觉避障
├── 单目/双目相机
├── AI 目标检测 (线缆/拖鞋/袜子/粪便)
├── 结构光 3D 感知
└── 优点: 识别障碍物类型; 缺点: 计算量大

第 4 代: 3D ToF + AI
├── dToF 传感器 (0.1-5m)
├── AI 芯片 (NPU)
├── 实时语义分割
└── 优点: 精确测距+识别; 缺点: 成本高

AI 避障模型:
├── 输入: 相机图像 + ToF 深度图
├── 模型: MobileNet / EfficientNet-Lite
├── 输出: 障碍物类型 + 位置 + 可通行性
├── 推理: <30ms (NPU)
└── 类别: 线缆/拖鞋/袜子/玩具/宠物/粪便/家具
```

### 5. 拖地系统

```
扫拖一体技术:

振动拖地:
├── 原理: 高频振动拖布 (1000-3000 次/分)
├── 优点: 去污力强
├── 缺点: 噪音大, 结构复杂
└── 代表: 追觅 S20

旋转拖地:
├── 原理: 双盘旋转 (150-200 RPM)
├── 优点: 清洁力强, 无死角
├── 缺点: 边角清洁差
└── 代表: 科沃斯 T30

拖布自清洁:
├── 基站清洗: 自动回洗拖布
├── 热风烘干: 防止细菌滋生
├── 自动上下水: 连接水管
├── 银离子除菌: 抑菌
└── 尘袋/尘盒: 自动集尘

水泵控制:
├── 电磁泵: 开关控制, 出水量固定
├── 蠕动泵: PWM 控制, 出水量可调
├── 出水量: 3 级可调 (低/中/高)
└── 渗水控制: 拖布湿度传感器
```
