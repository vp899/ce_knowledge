## 核心内容

### 1. 产品概述

| 项目 | 内容 |
|------|------|
| 产品名称 | SolarSail X1 |
| 产品定位 | 太阳能自主巡航无人船平台 |
| 目标用户 | 环保监测、水利巡检、科研采样 |
| 价格区间 | ¥29,800-49,800 |

### 2. 产品规格

### 船体参数
| 参数 | 规格 |
|------|------|
| 船体尺寸 | 1200×600×350mm |
| 船体重量 | 15kg (不含载荷) |
| 最大载荷 | 10kg |
| 最大速度 | 5m/s (约 10 节) |
| 巡航速度 | 1.5m/s (约 3 节) |
| 续航时间 | 太阳能: 无限 (晴天) / 电池: 8h |
| 防水等级 | IP67 |
| 抗浪等级 | 3 级 |

### 动力系统
| 参数 | 规格 |
|------|------|
| 推进方式 | 双螺旋桨 (差速转向) |
| 电机 | 无刷防水电机 ×2 |
| 电池 | 100Ah 磷酸铁锂 |
| 太阳能板 | 200W 单晶硅 (可折叠) |
| 充电 | 太阳能 + 岸基充电 |

### 导航系统
| 参数 | 规格 |
|------|------|
| 定位 | RTK-GNSS (厘米级) |
| 避障 | 毫米波雷达 + 超声波 |
| 航点导航 | 支持 100+ 航点 |
| 自主返航 | 低电量/失联自动返航 |
| 遥控距离 | 2km (遥控器) / 4G (无限) |

### 3. 载荷接口

| 接口 | 用途 |
|------|------|
| 水质传感器 | pH/溶解氧/浊度/温度 |
| 采样器 | 自动水样采集 (4-6 瓶) |
| 声呐 | 水深/水底地形 |
| 摄像头 | 水面监控/录制 |
| 扩展 RS485 | 第三方传感器 |

### 4. 验收标准

| 测试项 | 标准 |
|--------|------|
| 航行精度 | ±0.5m (RTK) |
| 续航 | ≥8h (纯电池), ≥24h (太阳能) |
| 防水 | IP67 (30 分钟浸水) |
| 避障 | 5m 内障碍物检测 |
| 通信 | 4G/遥控 2km |
