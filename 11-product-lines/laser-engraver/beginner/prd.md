## 核心内容

### 1. 产品概述

| 项目 | 内容 |
|------|------|
| 产品名称 | MakerLaser X1 |
| 产品定位 | 桌面级智能激光雕刻机 |
| 目标用户 | 创客、小型工作室、教育机构 |
| 价格区间 | ¥2,999-3,999 |
| 目标销量 | 首年 50,000 台 |

### 2. 产品规格

### 核心参数
| 参数 | 规格 |
|------|------|
| 激光类型 | 二极管激光 (蓝光 450nm) |
| 激光功率 | 20W (光输出) |
| 雕刻面积 | 400×400mm |
| 雕刻速度 | ≤20,000mm/min |
| 定位精度 | ±0.01mm |
| 最小线宽 | 0.04mm |
| 分辨率 | 1000 DPI |
| 支持材料 | 木材/皮革/亚克力/纸张/织物/不锈钢(打标) |

### 功能需求
| 优先级 | 功能 | 描述 |
|--------|------|------|
| P0 | 自动对焦 | 激光自动测量距离并调焦 |
| P0 | 安全防护 | 封闭式设计, 烟雾过滤 |
| P0 | WiFi 连接 | 手机/电脑无线控制 |
| P1 | 旋转雕刻 | 圆柱体雕刻附件 |
| P1 | 摄像头定位 | 拍照预览雕刻位置 |
| P2 | AI 设计 | 输入文字/图片自动生成雕刻文件 |
| P2 | 批量模式 | SD 卡离线运行 |

### 软件需求
| 功能 | 描述 |
|------|------|
| 设计软件 | 支持 SVG/PNG/JPG/DXF 导入 |
| 参数预设 | 木材/皮革/亚克力等材质参数库 |
| 实时预览 | 摄像头预览 + 区域选择 |
| 批量生产 | 数组排列, 序列号生成 |
| 手机 APP | iOS/Android 远程控制 |

### 3. 验收标准

| 测试项 | 标准 |
|--------|------|
| 雕刻精度 | ±0.05mm (100mm 范围内) |
| 重复精度 | ±0.02mm |
| 最大连续工作 | ≥24 小时 |
| 安全认证 | CE/FCC/Class 1 激光安全 |
| 噪音 | <55dB (不含排风) |
