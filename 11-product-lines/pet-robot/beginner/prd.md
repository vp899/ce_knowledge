## 核心内容

### 1. 产品概述

| 项目 | 内容 |
|------|------|
| 产品名称 | PetPal X1 |
| 产品定位 | 移动式宠物陪伴机器人 |
| 目标用户 | 独居上班族/宠物主人 |
| 价格区间 | ¥999-1,999 |

### 2. 产品规格

| 参数 | 规格 |
|------|------|
| 外形 | 圆形, 直径 20cm, 高 15cm |
| 移动 | 两轮驱动, 最大速度 0.5m/s |
| 摄像头 | 1080p 广角, 夜视 |
| 音频 | 双向语音, 扬声器 |
| 互动 | 激光逗猫, 零食投喂 |
| 续航 | 4h (持续移动), 24h (待机) |
| 充电 | 自动回充 |
| 连接 | WiFi + 蓝牙 |
| APP | iOS/Android |

### 3. 核心功能

| 优先级 | 功能 |
|--------|------|
| P0 | 远程视频查看/语音通话 |
| P0 | APP 遥控移动 |
| P0 | 自动巡逻 (定时) |
| P0 | 自动回充 |
| P1 | 激光逗猫 (自动/手动) |
| P1 | 零食投喂 (远程触发) |
| P1 | 宠物活动记录 |
| P2 | 宠物情绪识别 |
| P2 | 宠物异常行为报警 |
| P2 | 自动跟随宠物 |
