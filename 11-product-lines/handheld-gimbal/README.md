## 核心技术差异 (vs 无人机云台)

| 维度 | 无人机云台 | 手持云台 |
|------|-----------|---------|
| 振动源 | 电机高频振动 | 人手低频抖动 (1-10Hz) |
| 负载变化 | 固定相机 | 可换镜头/手机 |
| 姿态范围 | ±30° | ±360° (无限旋转) |
| 控制输入 | 飞控姿态 | 手柄按键/触摸/手势 |
| 功耗预算 | 充足 (电池供电) | 严格 (续航要求) |
| 交互方式 | APP 遥控 | 实体按键+屏幕 |
| 防护等级 | IP43+ | 一般 IP54 |
| 成本敏感度 | 中 | 高 (消费级) |

## 手持云台关键技术

### 1. 人手抖动模型
```
人手抖动频率分布:
├── 低频 (0.5-3Hz): 身体摆动, 幅度大
├── 中频 (3-8Hz): 手臂抖动, 幅度中
└── 高频 (8-12Hz): 手指微颤, 幅度小

典型抖动幅度:
├── 水平: ±2-5°
├── 垂直: ±1-3°
└── 旋转: ±1-2°

控制要求:
├── 抑制带宽: 0.1-15Hz
├── 残余抖动: <0.1° (RMS)
├── 响应延迟: <5ms
└── 跟踪平滑: 低通滤波 + 前馈
```

### 2. 手持云台电机选型
```
手持 vs 无人机电机差异:

手持云台电机特点:
├── KV 值更低 (20-80 KV): 需要更大扭矩
├── 槽极配合: 12N14P 或 9N12P (低齿槽)
├── 磁钢: 钕铁硼 N52 (高磁能积)
├── 绕线: 细线多匝 (高反电动势)
└── 轴承: 陶瓷轴承 (低摩擦)

选型表 (手持云台):
| 承重 | 电机型号 | KV | 扭矩 | 重量 | 典型产品 |
|------|----------|-----|------|------|----------|
| <200g | 1804 | 60 | 0.05Nm | 15g | 口袋相机 |
| 200-500g | 2204 | 50 | 0.15Nm | 25g | 手机云台 |
| 500g-1kg | 2804 | 40 | 0.3Nm | 40g | 运动相机 |
| 1-3kg | 3508 | 30 | 0.8Nm | 80g | 微单云台 |
| 3-10kg | 4010 | 20 | 2.0Nm | 150g | 专业云台 |
```

### 5. 手持云台 vs 无人机云台控制差异
```
手持云台控制特殊需求:

1. 人体运动补偿
   ├── 检测步行频率 (1-2Hz)
   ├── 自适应滤波 (步态感知)
   └── 平滑过渡 (运动/静止切换)

2. 多目标跟踪
   ├── 人脸检测 + 跟踪
   ├── 手势识别控制
   └── 构图辅助 (三分法)

3. 低功耗设计
   ├── 休眠模式 (<10mW)
   ├── 快速唤醒 (<500ms)
   ├── 动态频率调整
   └── 电池管理 (充放电保护)

4. 热管理
   ├── 连续工作温升控制
   ├── 降功率保护
   └── 用户感知温度 (<45°C)
```
