### 2. 产品规格

### 云台参数
| 参数 | 规格 |
|------|------|
| 轴数 | 三轴 (Roll/Pitch/Yaw) |
| 承重 | 150-300g |
| 续航 | 15 小时 |
| 充电 | USB-C, 2h 充满 |
| 重量 | 400g (含电池) |
| 折叠 | 可折叠便携 |
| 控制 | 摇杆+按键+手势 |

### 增稳性能
| 参数 | 规格 |
|------|------|
| 增稳范围 | ±0.01° 精度 |
| 最大转速 | 120°/s |
| 跟踪模式 | 脸部/人体/物体跟踪 |
| 全景模式 | 360°/240°/180° |
| 延时模式 | 移动延时/静态延时 |
| 盗梦空间 | Roll 轴连续旋转 |

### 智能功能
| 优先级 | 功能 |
|--------|------|
| P0 | 三轴增稳 |
| P0 | 人脸/人体跟踪 |
| P0 | 手势控制 (拍照/录像/跟踪) |
| P1 | 一键全景 |
| P1 | 移动延时 |
| P1 | 盗梦空间 |
| P2 | 希区柯克变焦 |
| P2 | AI 构图辅助 |

### 3. 验收标准

| 测试项 | 标准 |
|--------|------|
| 增稳精度 | <0.05° RMS |
| 续航 | ≥12h (连续使用) |
| 噪音 | <30dB (电机) |
| 跟踪成功率 | >95% (正常光照) |
| 折叠寿命 | >10,000 次 |
