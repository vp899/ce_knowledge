### 2. 产品规格

### 飞行性能
| 参数 | 规格 |
|------|------|
| 起飞重量 | 249g (免注册) |
| 最大飞行时间 | 35 分钟 |
| 最大飞行距离 | 15km |
| 最大飞行速度 | 58km/h (16m/s) |
| 最大抗风等级 | 5 级 (10m/s) |
| 最大飞行高度 | 500m (限高 120m) |
| 工作温度 | -10°C ~ 40°C |

### 相机系统
| 参数 | 规格 |
|------|------|
| 传感器 | 1/1.3 英寸 CMOS |
| 有效像素 | 4800 万 |
| 视频分辨率 | 4K@60fps / 2.7K@100fps |
| 照片分辨率 | 8000×6000 |
| 镜头 | 等效 24mm, f/1.7 |
| 云台 | 三轴机械增稳 |
| 变焦 | 数字变焦 4× |
| HDR | 照片 + 视频 |

### 图传系统
| 参数 | 规格 |
|------|------|
| 图传协议 | OcuSync 4.0 |
| 传输距离 | 20km (FCC) / 10km (CE) |
| 分辨率 | 1080p@60fps |
| 延迟 | <120ms |
| 频段 | 2.4GHz / 5.8GHz |

### 智能功能
| 优先级 | 功能 | 描述 |
|--------|------|------|
| P0 | 避障 | 四向避障 (前/后/下/左右) |
| P0 | GPS 定位 | GPS+GLONASS+北斗 |
| P0 | 自动返航 | 低电量/失联返航 |
| P0 | 一键起飞/降落 | |
| P1 | 智能跟踪 | ActiveTrack 5.0 |
| P1 | 一键短片 | 渐远/螺旋/彗星/小行星 |
| P1 | 全景拍摄 | 球形/广角/180° |
| P2 | 延时摄影 | 定时/自由/环绕 |
| P2 | 夜景模式 | 长曝光合成 |
| P2 | 手势控制 | 手势拍照/跟随 |

### 遥控器
| 参数 | 规格 |
|------|------|
| 屏幕 | 5.5 英寸 1080p |
| 续航 | 4 小时 |
| 重量 | 350g |
| 操控杆 | 可拆卸 |

### 3. 验收标准

| 测试项 | 标准 |
|--------|------|
| 续航 | ≥30 分钟 (无风, 悬停) |
| 图传距离 | ≥10km (开阔地) |
| 定位精度 | ±1.5m (GPS) / ±0.5m (RTK) |
| 避障 | 15km/h 速度下检测障碍物 |
| 噪音 | <65dB (1m 处) |
| 抗风 | 5 级风稳定飞行 |
| 认证 | CE/FCC/SRRC/MIC |
