## 核心内容

### 1. 产品概述

| 项目 | 内容 |
|------|------|
| 产品名称 | AquaBot P1 |
| 产品定位 | 智能无线泳池清洁机器人 |
| 目标用户 | 别墅业主/酒店/健身房 |
| 价格区间 | ¥2,999-5,999 |

### 2. 产品规格

| 参数 | 规格 |
|------|------|
| 清洁方式 | 刷洗 + 吸污 |
| 清洁面积 | 最大 100m² |
| 清洁时间 | 1.5-3h (可选模式) |
| 过滤精度 | 2μm (超细滤网) |
| 电池 | 5000mAh 锂电池 |
| 续航 | 2-3h |
| 充电 | 无线充电座 |
| 爬墙 | 支持 (池壁清洁) |
| 适用泳池 | 混凝土/瓷砖/乙烯基 |
| 防水等级 | IPX8 |
| 重量 | 8kg |
| 遥控 | APP (WiFi/蓝牙) |

### 3. 核心功能

| 优先级 | 功能 |
|--------|------|
| P0 | 自动清洁 (池底+池壁+水线) |
| P0 | 智能路径规划 (全覆盖) |
| P0 | 低电量自动回充电座 |
| P0 | 防缠绕 (头发/绳索) |
| P1 | APP 控制 (定时/区域/模式) |
| P1 | 水质监测 (pH/温度) |
| P2 | 自动投放消毒剂 |
