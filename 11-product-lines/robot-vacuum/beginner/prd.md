## 核心内容

### 1. 产品概述

| 项目 | 内容 |
|------|------|
| 产品名称 | CleanBot X1 |
| 产品定位 | 全能 AI 扫拖机器人 |
| 目标用户 | 城市白领/家庭用户 |
| 价格区间 | ¥2,999-4,999 |

### 2. 产品规格

### 清洁性能
| 参数 | 规格 |
|------|------|
| 吸力 | 8000Pa |
| 扫地方式 | 双旋转边刷 + 胶刷主刷 |
| 拖地方式 | 双盘旋转拖布 (200RPM) |
| 拖布加压 | 10N 向下压力 |
| 清洁面积 | 200m² (单次) |
| 尘盒容量 | 400ml |
| 水箱容量 | 80ml (机载) |

### 导航与避障
| 参数 | 规格 |
|------|------|
| 导航 | LDS LiDAR + 视觉 SLAM |
| 避障 | 3D 结构光 + AI 识别 |
| 建图 | 多楼层地图 (5 层) |
| 定位精度 | ±2cm |
| 避障物体 | 线缆/拖鞋/袜子/宠物 |

### 全能基站
| 参数 | 规格 |
|------|------|
| 自动集尘 | 2.5L 尘袋 (60天) |
| 热水洗拖布 | 60°C 热水 |
| 热风烘干 | 拖布自动烘干 |
| 自动上下水 | 可选连接水管 |
| 银离子除菌 | 拖布除菌 |

### 续航
| 参数 | 规格 |
|------|------|
| 电池 | 5200mAh |
| 续航 | 180 分钟 (标准模式) |
| 充电 | 自动回充 + 断点续扫 |

### 3. 核心功能

| 优先级 | 功能 |
|--------|------|
| P0 | 扫拖一体清洁 |
| P0 | LDS+视觉 SLAM 建图 |
| P0 | AI 避障 (线缆/拖鞋/粪便) |
| P0 | 自动集尘+洗拖布+烘干 |
| P1 | APP 控制 (区域/禁区/定时) |
| P1 | 语音控制 (小爱/Alexa) |
| P1 | 多楼层地图 |
| P2 | 自动上下水 |
| P2 | 视频监控 (巡航拍照) |
