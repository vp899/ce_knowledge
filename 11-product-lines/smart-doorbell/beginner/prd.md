### 1. 产品概述

| 项目 | 内容 |
|------|------|
| 产品名称 | SmartBell X1 |
| 产品定位 | AI 智能可视门铃 |
| 目标用户 | 家庭用户/租房用户 |
| 价格区间 | ¥299-499 |

### 2. 产品规格

| 参数 | 规格 |
|------|------|
| 摄像头 | 2K (2560×1440) |
| 视角 | 160° 超广角 |
| 夜视 | 红外夜视 (10m) |
| AI 识别 | 人脸/包裹/宠物/车辆 |
| 门铃按钮 | 触摸式 + LED 指示 |
| 双向语音 | 全双工, 降噪 |
| 存储 | microSD (最大 256GB) + 云 |
| 供电 | 内置电池 (6 个月续航) / 有线 |
| 连接 | WiFi 6 (2.4GHz) |
| 安装 | 3M 背胶 / 螺丝 |
| 防水 | IP65 |
| 工作温度 | -20°C ~ 50°C |
