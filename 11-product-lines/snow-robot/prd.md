# 除雪机器人 PRD

## 1. 产品概述

| 项目 | 内容 |
|------|------|
| 产品名称 | SnowBot S1 |
| 产品定位 | 家用自主除雪机器人 |
| 目标用户 | 北方家庭/别墅/小型物业 |
| 价格区间 | ¥4,999-6,999 |

## 2. 产品规格

| 参数 | 规格 |
|------|------|
| 除雪宽度 | 500mm |
| 除雪深度 | 最大 200mm |
| 除雪速度 | 0.5-1.5m/s |
| 除雪方式 | 螺旋桨收集 + 抛雪 (最远 5m) |
| 驱动方式 | 履带驱动 (防滑) |
| 电池 | 48V/20Ah 锂电池 |
| 续航 | 2-3h (约 500-1000m²) |
| 自主导航 | RTK + 边界线 |
| 防水等级 | IP65 |
| 工作温度 | -25°C ~ 5°C |
| 重量 | 45kg |

## 3. 核心功能
| 优先级 | 功能 |
|--------|------|
| P0 | 自动定时除雪 (天气联动) |
| P0 | 边界检测 (虚拟围栏) |
| P0 | 避障 (石头/台阶/宠物) |
| P0 | 低电量自动回充 |
| P1 | APP 远程控制 |
| P1 | 多区域规划 |
| P2 | 割草模式 (春季) |
