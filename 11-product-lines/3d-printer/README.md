# 3D 打印机产品知识库

## 产品分类

```
3D 打印机技术路线:

├── FDM (熔融沉积建模)
│   ├── 原理: 热塑性丝材加热挤出, 逐层堆叠
│   ├── 精度: ±0.1-0.3mm
│   ├── 材料: PLA/ABS/PETG/TPU/PA
│   ├── 成本: ¥1000-5000 (桌面级)
│   └── 代表: Creality Ender 3, Bambu Lab P1S
│
├── SLA/DLP (光固化)
│   ├── 原理: 紫外光固化树脂, 逐层曝光
│   ├── 精度: ±0.02-0.05mm
│   ├── 材料: 光敏树脂
│   ├── 成本: ¥2000-10000
│   └── 代表: Anycubic Photon, Formlabs Form 3
│
├── SLS (选择性激光烧结)
│   ├── 原理: 激光烧结粉末材料
│   ├── 精度: ±0.1mm
│   ├── 材料: 尼龙/金属粉末
│   ├── 成本: ¥50000+
│   └── 代表: Formlabs Fuse 1
│
└── 其他
    ├── MJF (多射流熔融): HP, 工业级
    ├── EBM (电子束熔融): 金属打印
    └── Binder Jetting: 粘结剂喷射
```

## FDM 打印机核心系统

### 1. 机械结构
```
FDM 打印机运动结构:

结构 1: Cartesian (笛卡尔)
┌─────────────────────────────┐
│   X 轴 (水平)              │
│   ┌───────────────────┐    │
│   │   喷头            │    │
│   └───────────────────┘    │
│        │                    │
│   Y 轴 (平台前后)          │
│   ┌───────────────────┐    │
│   │   热床            │    │
│   └───────────────────┘    │
│                             │
│   Z 轴 (垂直升降)          │
│   │                        │
│   ▼                        │
└─────────────────────────────┘
代表: Ender 3, Prusa i3

结构 2: CoreXY
┌─────────────────────────────┐
│   X/Y 轴共用两个电机       │
│   ┌───────────────────┐    │
│   │   喷头 (X+Y 联动) │    │
│   └───────────────────┘    │
│                             │
│   Z 轴 (平台升降)          │
│   ┌───────────────────┐    │
│   │   热床            │    │
│   └───────────────────┘    │
└─────────────────────────────┘
优点: 高速, 高精度
代表: Bambu Lab X1, Voron 2.4

结构 3: Delta (三角洲)
    三个垂直轴
    │   │   │
    │   │   │
    ▼   ▼   ▼
   ┌─────────┐
   │  喷头   │ (三轴联动)
   └─────────┘
优点: 高速, 大打印高度
缺点: 精度略差
代表: Anycubic Kossel
```

### 2. 挤出系统
```
挤出机 (Extruder):

直接驱动 (Direct Drive):
┌─────────────────┐
│  步进电机       │
│  ┌───────────┐ │
│  │ 齿轮组    │ │  ← 近程挤出
│  │ (减速比3:1)│ │
│  └─────┬─────┘ │
│        │       │
│    丝材入口     │
│        │       │
│   ┌────┴────┐  │
│   │ 热端    │  │  ← 加热块 + 喷嘴
│   │ (加热+  │  │
│   │  散热)  │  │
│   └────┬────┘  │
│        │       │
│      喷嘴      │  ← 0.4mm 喷嘴
└─────────────────┘
优点: 回抽精准, 软材友好
缺点: 打印头重, 速度受限

远程挤出 (Bowden):
步进电机 ──── 管道 ──── 打印头
(机身)      (PTFE管)    (轻量)
优点: 打印头轻, 高速
缺点: 回抽延迟, 软材困难

热端参数:
├── 加热功率: 40-60W
├── 最高温度: 260°C (标准) / 300°C (高温)
├── 喷嘴直径: 0.2/0.4/0.6/0.8mm
├── 热断: 全金属 / PTFE 衬管
└── 热敏: NTC 100K (精度 ±1°C)
```

### 3. 固件 (Marlin / Klipper)

```
Marlin 固件架构:
├── 核心: Arduino 框架
├── 平台: AVR / STM32 / ESP32
├── 特点: 成熟稳定, 社区大
└── 缺点: 配置复杂, 高速受限

Klipper 固件架构:
├── 核心: Linux + MCU 协处理
├── 主机: Raspberry Pi / 类似 SBC
├── MCU: STM32 (步进电机控制)
├── 特点: 高速, 配置灵活, 输入整形
└── 优点: 32 位精度, 多轴支持

Klipper 配置示例:
[stepper_x]
step_pin: PF13
dir_pin: PF12
enable_pin: !PF14
microsteps: 16
rotation_distance: 40
full_steps_per_rotation: 200
max_velocity: 300
max_accel: 3000

[extruder]
step_pin: PA4
dir_pin: !PA10
enable_pin: !PA2
microsteps: 16
rotation_distance: 33.500
nozzle_diameter: 0.400
filament_diameter: 1.750
heater_pin: PB4
sensor_type: ATC Semitec 104NT-4-R025H42G
sensor_pin: PK0
min_temp: 0
max_temp: 260
max_power: 1.0

[input_shaper]
shaper_type: mzv
shaper_freq_x: 57.0
shaper_freq_y: 48.0
```

### 4. 切片软件 (Slicer)

```
切片流程:
3D 模型 (STL/3MF)
    │
    ▼
┌─────────────────┐
│   导入模型      │
│   (网格修复)     │
├─────────────────┤
│   设置参数      │
│   ├── 层高      │
│   ├── 壁厚      │
│   ├── 填充率    │
│   ├── 支撑      │
│   └── 温度/速度 │
├─────────────────┤
│   切片计算      │
│   (G-code 生成) │
├─────────────────┤
│   路径优化      │
│   ├── 回抽      │
│   ├── 调速      │
│   └── 冷却      │
└─────────────────┘
    │
    ▼
G-code 输出

切片参数对质量影响:
| 参数 | 小值 | 大值 | 影响 |
|------|------|------|------|
| 层高 | 0.08mm | 0.3mm | 精度 vs 速度 |
| 壁厚 | 0.8mm | 2mm | 强度 vs 材料 |
| 填充率 | 10% | 100% | 强度 vs 重量 |
| 打印速度 | 30mm/s | 300mm/s | 质量 vs 时间 |
| 温度 | 190°C | 230°C (PLA) | 层粘 vs 拉丝 |
| 回抽距离 | 1mm | 8mm | 拉丝 vs 堵头 |

G-code 基础指令:
G28        ; 归零
G29        ; 自动调平
G1 X100 Y100 Z0.3 F3000  ; 移动
M104 S200  ; 设置喷头温度
M140 S60   ; 设置热床温度
M109 S200  ; 等待喷头温度
M190 S60   ; 等待热床温度
G1 E5 F300 ; 挤出 5mm
M106 S255  ; 风扇全速
M107       ; 关闭风扇
```

### 6. 热床与调平

```
自动调平技术:

ABL (Auto Bed Leveling):
├── 探针类型: BLTouch (伺服+探针), CRTouch (光学)
├── 探测点: 3×3 ~ 16×16 网格
├── 补偿: 软件 Z 偏移补偿
└── 精度: ±0.02mm

Klippain (Klipper 自动调平):
├── 传感器: 应变片 / 电容式
├── 探测: 25-100 点
├── 算法: 最小二乘拟合
└── 输出: 倾斜补偿 + 网格补偿

调平 G-code:
G28           ; 归零
BED_MESH_CALIBRATE  ; Klipper 自动调平
BED_MESH_OUTPUT     ; 输出网格数据
BED_MESH_PROFILE SAVE=default  ; 保存配置

热床类型:
├── PCB 热床: 平整, 加热均匀, 成本低
├── 铝板热床: 导热好, 需贴膜
├── 弹簧钢板: 可弯曲脱模, 磁吸固定
└── 玻璃床: 平整度最高, 需涂胶
```
