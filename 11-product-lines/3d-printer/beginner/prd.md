## 核心内容

### 1. 产品概述

| 项目 | 内容 |
|------|------|
| 产品名称 | PrintX1 |
| 产品定位 | 高速智能 3D 打印机 |
| 目标用户 | 创客/设计师/教育 |
| 价格区间 | ¥2,499-4,999 |

### 2. 产品规格

| 参数 | 规格 |
|------|------|
| 打印技术 | FDM (熔融沉积) |
| 运动结构 | CoreXY |
| 打印尺寸 | 256×256×256mm |
| 最大速度 | 500mm/s |
| 最大加速度 | 20,000mm/s² |
| 喷嘴直径 | 0.4mm (可换 0.2/0.6/0.8) |
| 层高 | 0.08-0.28mm |
| 材料 | PLA/ABS/PETG/TPU/PA/PC |
| 喷头温度 | 最高 300°C |
| 热床温度 | 最高 110°C |
| 调平 | 自动 (压力传感器) |
| 耗材直径 | 1.75mm |
| 连接 | WiFi/USB/SD 卡 |
| 显示 | 4.3" 触摸屏 |

### 3. 核心功能

| 优先级 | 功能 |
|--------|------|
| P0 | 高速打印 (500mm/s) |
| P0 | 自动调平 |
| P0 | 断电续打 |
| P0 | 断料检测 |
| P1 | Input Shaper (振动补偿) |
| P1 | 多色打印 (AMS) |
| P1 | WiFi 远程控制 |
| P2 | AI 首层检测 |
| P2 | 摄像头监控 |
