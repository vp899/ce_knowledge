# 07 - 可靠性测试

## 模块概述

消费电子产品可靠性测试体系：环境测试、机械测试、[[env-testing|EMC]] 测试、寿命评估。

## 目录结构

```
07-reliability/
├── environmental/  # 环境测试（温湿度、盐雾、防水）
├── mechanical/     # 机械测试（跌落、振动、按键寿命）
├── emc/            # EMC 测试（辐射、传导、ESD）
└── lifecycle/      # 寿命评估与加速老化
```

## 核心知识领域

### 1. 环境测试

#### 温湿度测试矩阵
| 测试项目 | 条件 | 时长 | 标准 |
|----------|------|------|------|
| 高温工作 | 55°C | 72h | IEC 60068-2-2 |
| 低温工作 | -20°C | 72h | IEC 60068-2-1 |
| 高温高湿 | 85°C/85%RH | 1000h | IEC 60068-2-78 |
| [[env-testing|温度循环]] | -40°C~85°C, 30min/cycle | 500 cycles | IEC 60068-2-14 |
| 快速温变 | 15°C/min | 100 cycles | JESD22-A104 |
| 盐雾测试 | 5% NaCl, 35°C | 48-96h | IEC 60068-2-11 |
| 防水测试 | IPX7 (1m/30min) | - | IEC 60529 |

#### IP 防护等级
| 等级 | 防尘 | 防水 |
|------|------|------|

### 3. EMC 测试

#### 测试项目与标准
| 项目 | 标准 | 限值 |
|------|------|------|
| 辐射发射 (RE) | C[[camera-sensor|ISP]]R 32 | Class B: 30dBμV/m@3m |
| 传导发射 (CE) | CISPR 32 | Class B: 46dBμV |
| 静电放电 (ESD) | IEC 61000-4-2 | ±8kV 接触 / ±15kV 空气 |
| 浪涌 (Surge) | IEC 61000-4-5 | ±1kV 线对线 |
| 传导抗扰 (CS) | IEC 61000-4-6 | 3V, 150kHz-80MHz |
| 辐射抗扰 (RS) | IEC 61000-4-3 | 3V/m, 80MHz-6GHz |
| 电压跌落 | IEC 61000-4-11 | 0%, 1周期 |

#### EMC 整改思路
```
问题定位 → 频谱分析 → 路径识别 → 措施选择 → 验证
    │                                          │
    ├── 滤波 (共模/差模电感、磁珠、电容)        │
    ├── 屏蔽 (金属壳体、导电泡棉)              │
    ├── 接地 (单点/多点接地、搭接)              │
    ├── 布局 (高频器件远离接口)                 │
    └── PCB (完整地平面、过孔缝合)              │
```

### 4. 寿命评估

#### 加速老化模型
| 模型 | 公式 | 适用 |
|------|------|------|
| Arrhenius | AF = exp[Ea/k × (1/T_use - 1/T_test)] | 温度加速 |
| Coffin-Manson | AF = (ΔT_test/ΔT_use)^n | 热循环 |
| Eyring | 多应力联合 | 温度+湿度+电压 |
| 威布尔分布 | F(t) = 1-exp(-(t/η)^β) | 寿命预测 |

#### [[mechanical-testing|MTBF]] 目标
| 产品类型 | MTBF 目标 | 测试时长 |
|----------|-----------|----------|
| 消费电子 | >30,000h | 1000h |
| 工业设备 | >100,000h | 3000h |
| 汽车电子 | >150,000h | 5000h |
| 医疗设备 | >200,000h | 8000h |
